## Hardware

I have 3-4 years old dual-booted **Asus Vivobook Pro 15** laptop, with following specs:

- RAM: 16GB
- SSD: 512GB (250GB for _arch_, remaining for windows)
- Nvidia GTX 1650

## Software

Below is a list of software I use day to day:

- _hyprland_ as my desktop environment.
- _alacritty_ as my standard terminal, I also use kitty sometimes.
- _Nvim_ with _Nvchad_ as my editor.
- _pandoc_ for converting markdown to pdfs.
- Google Chrome as web-browser.
- _zathura_ for pdf viewing.
- _yazi_ as file-manager, with _kitty_ to see previews.
- I also love to use *tui*s like _lazygit_ and _lazydocker_.

> You can find my config files [here](https://github.com/sayymeer/dotfiles)
